Fake News Detection API Documentation

Endpoint

POST /api/predict/

Request Format

Use the following format to send a request to the Fake News Detection API:

{
    "text": "Your news content here"
}

Parameters

  • text: String - The news article text you want to analyze. This parameter is required.

Response Format

The API returns a JSON response with the following structure:

{
    "text": "Your news content here",
    "prediction": "Real" or "Fake",
    "confidence": 0.97,
    "facts_checked": "Facts information here"
}

Response Fields

  • text: String - The original news text that was analyzed.
  • prediction: String - The result of the fake news detection. Either "Real" or "Fake".
  • confidence: Float - A confidence score (0-1) indicating the certainty of the prediction.
  • facts_checked: String - Any fact-checked information related to the news content.

Example Request

curl -X POST "http://127.0.0.1:8000/api/predict/" -H "Content-Type: application/json" -d '{
    "text": "Breaking news content..."
}'

Example Response

{
    "text": "Breaking news content...",
    "prediction": "Fake",
    "confidence": 0.89,
    "facts_checked": "Fact-check details here..."
}
